# Hardware Setup

OpenCue is designed with professional teleprompter glass (like the Elgato Prompter) in mind.

## Using an External Display
1. Connect your external monitor or Elgato Prompter to your Mac.
2. In OpenCue, open your script in the Editor.
3. Click the **External** display button in the toolbar and select the target monitor.
4. OpenCue will project the teleprompter full-screen to that monitor, while replacing your editor with the **Operator Controls View**.

## Mirroring Text
If you are using beam-splitter glass, the text might appear backward.
1. Click **Settings** in the Editor.
2. Under Mirroring, toggle **Mirror Horizontal**.
3. Now the external output will be perfectly readable through the glass!
